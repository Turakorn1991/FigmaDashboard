## สิ่งที่ทำในเซสชันนี้

### 1. Thai DatePicker (วันที่อนุญาต เริ่ม / สิ้นสุด)
- เปลี่ยนจาก `<input type="text">` เป็น component `ThaiDatePicker` แบบ custom inline
- เปิด popup ปฏิทินเมื่อคลิก
- Header แสดงชื่อเดือนภาษาไทย + ปี **พ.ศ.** (เช่น "มิถุนายน 2569")
- วันในสัปดาห์: อา จ อ พ พฤ ศ ส
- แสดงค่าที่เลือกในรูปแบบ `วว/ดด/ปปปป` (พ.ศ.)
- กด X ล้างค่าได้ / คลิกนอก popup ปิดอัตโนมัติ
- เก็บค่าภายในเป็น `YYYY-MM-DD` (พ.ศ.)

---

### 2. คอลัมน์ "ประเภทขนย้าย"
- เพิ่มระหว่าง **วันที่หมดอายุ** และ **ผู้ประกอบการ**
- แสดงเป็นข้อความสีดำปกติ (ไม่มี badge)
- มี filter dropdown ในหัวตาราง
- ค่าที่เป็นไปได้:
  - ขนย้าย
  - ขายขนย้ายในราชอาณาจักร
  - ขายขนย้ายนอกราชอาณาจักร
- Mock data กระจายตาม `id % 3`
- **Export Excel** มีคอลัมน์นี้ด้วย (อยู่หลัง วันที่หมดอายุ)

---

### 3. Format เลขที่หนังสือ อ.10
- เปลี่ยนจาก `อ.10-2568-0001` เป็น `001/2568`
- ปีเป็น พ.ศ. ตามที่เก็บในข้อมูล (ไม่บวก 543 เพิ่ม)

---

### 4. เลข # ต่อเนื่องข้ามหน้า
- เพิ่ม state `tablePage` + `tablePageSize`
- คำนวณ: `(tablePage - 1) * tablePageSize + i + 1`
- หน้า 2 = 11–20, หน้า 3 = 21–30 ฯลฯ
- กดค้นหา/รีเซ็ต → วนกลับหน้า 1 อัตโนมัติ
