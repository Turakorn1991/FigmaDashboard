### 5. Export Excel
- ชื่อไฟล์: **`ยอดการขนย้ายส่งมอบอาวุธหรือวัตถุ.xlsx`**
- คอลัมน์: เลขที่หนังสือ → วันที่อนุญาต → วันที่หมดอายุ → ประเภทขนย้าย → ผู้ประกอบการ → กลุ่มหน่วยผู้ซื้อ → หน่วยผู้ซื้อ → รหัสอาวุธ → ชื่ออาวุธ → จำนวนที่ได้รับอนุญาต → หน่วยนับ → วันที่ขนย้าย → ครั้งที่ขนย้าย → จำนวนที่ขนย้าย → หน่วยนับ
- **ตัดที่อยู่ต้นทาง และที่อยู่ปลายทางออก**
